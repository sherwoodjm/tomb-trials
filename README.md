# Tomb Trials

Explore a tomb as two different people. Switch between characters to work together. Solve puzzles to find the treasure—and the exit.

## Controls

| Action | Keys |
|---|---|
| Open/exit menu | Escape key |
| Movement | ↑←↓→ / WASD |
| Talk/Interact | Spacebar, Enter, Z |
| Switch Players | F |
